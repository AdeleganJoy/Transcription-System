# Transcription-System
This is a light weight but powerful transcription tool that coverts videos and audios into accurate transcriptions using the AI model whisper.

## Features
* **The system works with both audio and video files**
* **The system has a straight forward UI design, making it easy to use**

## Supported formats
- The system supports :
  - Audio (MP3, WAV, M4A (including zoom audios), FLAC, OGG).
  - Video (MP4, WEBM, MPEG).
  
## Installation

### Clone the Repository
```bash
git clone https://github.com/AdeleganJoy/Transcription-System.git
```

## Setup
### Create virtual environment
```bash
python3 -m venv .env
```
### Activate virtual environment (Mac/Linux)
```bash
source .env/bin/activate
```
### Activate virtual environment (Windows)
```bash
.env\Scripts\activate
```
### Install dependencies
```bash
pip install -r requirements.txt
```
## Run
```bash
python app.py
```
